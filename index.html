TDES M4 / EMDS M5

1- tag: < html > É a tag principal do documento. Todo o conteúdo da página fica dentro dela. Ela não aparece na tela. Apenas organiza toda a estrutura do site.

2- tag: < head > Guarda informações sobre a página, como título e configurações. Não aparece diretamente na página.

3- tag: < title > Define o título que aparece na aba do navegador. "Título da página"

4- tag: < meta > Define configurações importantes, como codificação de caracteres e responsividade. Não aparece na página. Funciona “por trás”.

5- tag: < h1 > até < h6 >, São tags de títulos e subtítulos. h1 é o maior e o h6 é o menor

6- tag: < p > Define um parágrafo de texto. O texto aparece separado em formato de parágrafo.

7- tag: < a >Cria um link para outra página ou site. Aparece como um texto clicável.

8- tag: < button > é usada para criar um botão clicável na página.

9- tag: < img >Exibe uma imagem na página. A imagem aparece na tela.

10- tag: < footer > Representa o rodapé da página. Fica no final da página, com informações extras.
