Explorando Tags Básicas

Com o Pedro Paulo!

tag: < html > É a tag principal do documento. Todo o conteúdo da página fica dentro dela. Ela não aparece na tela. Apenas organiza toda a estrutura do site.

tag: < head > Guarda informações sobre a página, como título e configurações. Não aparece diretamente na página.

tag: < title > Define o título que aparece na aba do navegador. "Título da página"

tag: < meta > Define configurações importantes, como codificação de caracteres e responsividade. Não aparece na página. Funciona “por trás”.

tag: < h1 > até < h6 >, São tags de títulos e subtítulos. h1 é o maior e o h6 é o menor

tag: < p > Define um parágrafo de texto. O texto aparece separado em formato de parágrafo.

tag: < a >Cria um link para outra página ou site. Aparece como um texto clicável.

tag: < button > é usada para criar um botão clicável na página.

tag: < img >Exibe uma imagem na página. A imagem aparece na tela.

tag: < footer > Representa o rodapé da página. Fica no final da página, com informações extras.
